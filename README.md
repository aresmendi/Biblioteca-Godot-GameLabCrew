# 🎮 Biblioteca Godot - GameLabCrew

Repositorio para almacenar, organizar y reutilizar **bibliotecas, sistemas y mecánicas** para videojuegos desarrollados con **Godot Engine**.

---

## 📌 Objetivo

Este repositorio sirve como una **base de recursos reutilizables** para acelerar el desarrollo de videojuegos, evitando reinventar la rueda en cada proyecto.

Incluye sistemas listos para usar en distintos tipos de juegos:

* 🧍‍♂️ 3ª persona
* 🔫 1ª persona
* 🧭 Isométricos
* 🎲 2D / 3D
* ⚙️ Sistemas generales reutilizables


## 🚀 ¿Qué puedes encontrar aquí?

* Controladores de personaje (FPS, TPS, etc.)
* Sistemas de cámara
* Mecánicas de movimiento
* Sistemas de combate
* Inventarios
* UI reutilizable
* Shaders
* Scripts útiles en GDScript

---

## 🤝 Contribuir

¡Este repositorio es colaborativo! Puedes contribuir de varias formas:

1. Haz un fork del repositorio
2. Crea una nueva rama:

   ```bash
   git checkout -b feature/nueva-funcionalidad
   ```
3. Añade tu sistema o mejora
4. Haz commit:

   ```bash
   git commit -m "Añadido sistema de X"
   ```
5. Haz push y abre un Pull Request

---

## 📏 Normas de contribución

* 📌 Código limpio y comentado
* 📁 Organizar bien las carpetas
* 🧩 Sistemas desacoplados y reutilizables
* 📄 Incluir un README dentro de cada módulo explicando su uso
* 🎮 Compatible (si es posible) con varias versiones de Godot

---

## 🧠 Filosofía

La idea es construir una **biblioteca propia del equipo**, optimizada para nuestros proyectos, fomentando la reutilización y la colaboración.
